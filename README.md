# 📄 Document Signature App

A full-stack web application for uploading, previewing, and digitally signing PDF documents.

## 🚀 Features

- User Registration & Login with JWT Authentication
- Upload PDF Documents
- Preview PDF files in browser
- Draw and save digital signatures
- Document status tracking (Pending / Signed / Rejected)
- Audit Trail (who signed, when, IP address)
- Responsive UI with Tailwind CSS

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Multer (file upload)
- Bcryptjs
